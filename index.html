LXP32 is a lightweight, open source 32-bit CPU IP core optimized for FPGA implementation

Features

Portable

  • Described in portable VHDL-93, not tied to any particular vendor
  • Compatible with later VHDL standard revisions

Optimized for FPGA

  • 3-stage pipeline
  • 256 registers implemented as block RAM (does not require distributed RAM)
  • Less than 30 distinct opcodes

Harvard architecture

  • Separate instruction and data buses
  • WISHBONE compatible
  • Can use on-chip or external RAM

Configurable

  • Three multiplier options
  • Optional instruction cache
  • Optional divider

Open source

LXP32 is distributed under the terms of the MIT license
